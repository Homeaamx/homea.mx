## Mapeo (Oxatis → 10 categorías de la guía)

Agrupación vigente. Algunos nodos Oxatis son transversales (Ofertas, Novedades, Más vendidos,
Stock, Outlet) → se modelan como **tags/colecciones automáticas**, no como categoría:

| Categoría (guía)               | Nodos Oxatis que la alimentan |
|--------------------------------|-------------------------------|
| **Cocina y Bar**               | EQUIPOS DE COCINA (todo: campanas, estufas, hornos, lavavajillas, microondas, parrillas, refrigeración, monomandos, tarjas, cajones calientes, cafeteras), Accesorios para Cocinar |
| **Exterior – Terraza**         | Asadores & Grills, Calentadores para Exterior, Nebulización para Terrazas, Jardín |
| **Baños**                      | BAÑOS (grifería para lavamanos/bidet/regaderas/tina, accesorios para baño). NO incluye tinas/jacuzzi/hot tubs → esos van a Wellness |
| **Vapor y Sauna**              | Sauna (Baño Sauna), Vapor Húmedo (Baño de Vapor) |
| **Recubrimientos & Superficies** | Superficies Sólidas |
| **Electrodomésticos menores**  | Electrodomésticos Menores, Otros (aspiradoras, enseres menores, dispensadores, despachadores de agua, calentadores de agua) |
| **Cuidado de ropa**            | Lavadoras & Secadoras, Planchas de Vapor |
| **Wellness**                   | Tinas de Inmersión, Carril de Nado, Sillón para Masaje, Jacuzzi de Exterior. Absorbe los nodos Oxatis "Hot Tubs" y "Tinas & Jacuzzi" (hoy bajo Baños) |
| **Chimeneas**                  | Chimeneas para Exterior & Interior (gas, leña, etanol, eléctricas) |
| **Minisplits**                 | Minisplits (Aire Acondicionado): 1.0 / 1.5 / 2.0 Tonelada |

Transversales (no son categoría → tag/colección): Ofertas y Promociones, Novedades, Productos
Más Vendidos, Productos en Stock, Outlet (Esencial/Premium), Listas de Precios.

> Notas:
> - "Campanas Profesionales (BORRAR)" → marcado para eliminar en Oxatis, NO migrar.
> - Filtros de Agua / Monomandos / Tarjas viven bajo Cocina y Bar (vienen de EQUIPOS DE COCINA en Oxatis).
