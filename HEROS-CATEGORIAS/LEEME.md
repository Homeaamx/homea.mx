# Heros de macrocategoría — buzón de imágenes

Sube aquí las fotos de escena para el hero de cada página de categoría
(originales, sin optimizar — de eso se encarga Claude: redimensiona,
comprime y les pone nombre SEO antes de integrarlas al preview).

## Nombre de archivo

Usa la slug de la categoría (cualquier extensión: jpg/png/webp):

| Archivo                              | Página                                          |
|--------------------------------------|-------------------------------------------------|
| `exterior.jpg`                       | categoria-exterior.html                         |
| `electrodomesticos-menores.jpg`      | categoria-electrodomesticos-menores.html        |
| `lavanderia.jpg`                     | categoria-lavanderia.html                       |
| `banos.jpg`                          | categoria-banos.html                            |
| `minisplits.jpg`                     | categoria-minisplits.html                       |
| `vapor-y-sauna.jpg`                  | categoria-vapor-y-sauna.html                    |
| `wellness.jpg`                       | categoria-wellness.html                         |
| `recubrimientos-y-superficies.jpg`   | categoria-recubrimientos-y-superficies.html     |
| `chimeneas-y-calentadores.jpg`       | categoria-chimeneas-y-calentadores.html         |

(Cocina y Bar ya tiene su hero interactivo Wolf/Sub-Zero.)

## Qué foto funciona mejor

- Horizontal, idealmente ~2000×1126 o mayor (proporción ≈ 16:9).
- Escena completa con piezas identificables (como la de Cocina y Bar):
  si los productos se distinguen, se pueden marcar los puntos
  interactivos (spotlights) sobre cada aparato.
- Zona baja de la imagen relativamente despejada: ahí va el título.
