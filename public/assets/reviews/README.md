# Fotos de perfil — Reseñas de Google (home)

Sube aquí la foto de cada reseñante. Si falta una imagen, la tarjeta muestra
automáticamente la inicial del nombre (no se rompe nada).

## Especificaciones
- **Formato:** .jpg o .webp (preferible .webp)
- **Forma:** cuadrada (se recorta a círculo). Mínimo 96×96 px, ideal 200×200 px.
- **Peso:** < 30 KB cada una (optimizar).
